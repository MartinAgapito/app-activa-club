# packages/validation

Esquemas de validación compartidos (Zod) entre `apps/web` y `apps/api`.

Este paquete extiende la configuración estricta compartida (`tsconfig.base.json`).
Aún no contiene esquemas: solo el esqueleto del paquete. Los esquemas reales
deben implementarse junto con las historias de usuario correspondientes y
mantenerse sincronizados con `packages/shared-types`.
